## Description

```
QIEPay is a full-stack decentralized payment gateway built on the QIE Blockchain (Testnet, Chain ID 1983). It enables merchants to accept crypto payments with a familiar, Stripe-like experience — no technical knowledge required.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✨ KEY FEATURES

💳 Payment Gateway
• Create payment invoices with amount & description
• Customers pay via QR code or wallet connect
• Email wallet support — no browser extension needed
• Real-time payment status tracking
• Self-pay detection & balance validation

📊 Merchant Dashboard
• Revenue analytics with interactive charts
• Transaction history with copy/view actions
• Earnings breakdown (QIE + USD equivalent)
• POS (Point of Sale) mode for in-store payments

🪙 DeFi Integration
• Staking — stake QIE tokens for fee tier discounts (Bronze → Gold)
• Governance — vote on protocol proposals
• Rewards — loyalty cashback system
• Batch payments — pay multiple recipients in one transaction

🔌 Developer Tools
• REST API with API key authentication
• Webhook notifications for payment events
• Embeddable payment widget for any website
• Network export (CSV) for accounting

🔐 Dual Wallet System
• Browser extension wallet (MetaMask-compatible)
• Email-based wallet (no extension required)
• Both support all features: payments, staking, governance

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

• Frontend: React + Vite + Tailwind CSS
• Backend: Vercel Serverless Functions
• Blockchain: QIE Testnet (Chain ID 1983)
• Smart Contracts: Solidity (4 deployed contracts)
  - QIEPay (Payment Gateway)
  - Staking
  - Governance
  - Rewards
```
